2. Prompt

2.1 Chaine de caractère

La fonction prompt s'utilise comme alert mais a une petite particularité. Elle renvoie ce que l'utilisateur a écrit sous forme d'une chaîne de caractères, voilà pourquoi on écrit de cette manière :

var text = prompt('Tapez quelque chose :');

Ci dessous un exemple, et appliqué sur cette page ( actualisez la page pour revoir l'exemple )

<script>
  var nom, prenom, result;

  nom = prompt('Tapez votre nom :');
  prenom = prompt('Très bien, votre prénom ensuite :');
  result = nom + ' ' + prenom; // le (' ') est utilisé pour ajouter d'espace.

  alert(result);
</script>
