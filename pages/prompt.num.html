2.2 Nombres ( addition )

Pour les nombres fournis par l'utilisateur, le résultat n'est pas très satisfaisante, pour ce fait, l'utilisation de la fonction parseInt() est nécessaire.

Voici un exemple de code :

<script>
  var nombre1, nombre2, result;

  nombre1 = prompt('Tapez le premier nombre :');
  nombre2 = prompt('Tapez le deuxième nombre :');
  result = parseInt(nombre1) + parseInt(nombre2);

  alert(result);
</script>

Les valeurs du fonction prompt() sont récuprés sous forme de chaine de caractère, la fonction parseInt() s'occupe de la conversion de ce dernier en nombre.
