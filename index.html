JAVASCRIPT

1. Alert

Cette page contient un alert dès l'ouverture de la page et après actualisation.

<script>
  alert("Contenu de l'alert!");
</script>

Il est aussi possible de l'afficher à l'aide d'un boutton, voici un exemple :

<script>
  function button() {
    alert('Ceci est un alert après clic du boutton !');
  }
</script>

<input type="button" value="Voir l'alert" onclick=button()>
